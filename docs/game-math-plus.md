Hãy tạo một mini game web giáo dục trẻ em từ 3 - 6 tuổi để học phép cộng bằng hình ảnh trực quan, phong cách giống game mobile kids learning cao cấp trên App Store/Google Play.

Mục tiêu:
- Giúp bé hiểu phép cộng bằng hình ảnh trực quan
- Học toán theo cách vui nhộn, dễ hiểu
- Tạo cảm giác như đang chơi game thay vì học

Phong cách thiết kế:
- Màu sắc cực kỳ tươi sáng
- Cute, glossy, sinh động
- UI giống app học toán cho trẻ em
- Bo góc lớn
- Shadow mềm
- Hiệu ứng phát sáng
- Animation mượt
- Có particle/confetti
- Touch-friendly cho tablet/mobile

Gameplay:
Ví dụ:
- Có 2 quả táo
- Thêm 3 quả táo nữa
- Kết quả bằng 5 quả táo

Hiển thị dạng:
[2 quả táo] + [3 quả táo] = [?]

Hoặc:
[2 quả táo] + [3 quả táo] = [5 quả táo]

Layout:
- 3 card lớn nằm ngang
- Card 1: nhóm object thứ nhất
- Card 2: nhóm object được thêm vào
- Card 3: kết quả
- Ở giữa có icon:
  - dấu "+"
  - dấu "="

Yêu cầu gameplay:
1. Random phép cộng:
   - phạm vi từ 1 → 10
2. Random object:
   - Táo
   - Cam
   - Chuối
   - Dâu
   - Kẹo
   - Bóng bay
   - Ngôi sao
   - Gấu bông
   - Xe đồ chơi
3. Có nhiều level:
   - Easy: 1 → 5
   - Medium: 1 → 10
   - Hard: 1 → 20
4. Có nhiều chế độ chơi:
   - Chọn đáp án đúng
   - Drag & Drop đáp án
   - Điền số còn thiếu
   - Đếm object rồi chọn kết quả
5. Có hệ thống điểm:
   - Score
   - Combo streak
   - Reward stars

Animation:
- Object mới sẽ:
   - bay vào
   - bounce nhẹ
   - pop animation
- Khi đúng:
   - confetti
   - sparkle
   - glow effect
   - mascot nhảy vui
- Khi sai:
   - rung nhẹ
   - hiện hint
   - phát âm thanh “Try again!”

Âm thanh:
- Background music nhẹ nhàng
- Voice đọc:
   - “Hai cộng ba bằng mấy?”
   - “Chính xác!”
   - “Giỏi lắm!”
   - “Thử lại nhé!”
- Sound effect:
   - pop
   - sparkle
   - success
   - click

UI chi tiết:
- Background thiên nhiên đẹp:
   - hồ nước
   - bầu trời
   - mây
   - cỏ xanh
- Balloon hiển thị số phía trên mỗi card
- Các card màu kem/trắng
- Nút màu tím glossy giống game trẻ em
- Font cực to, dễ đọc
- Các object rất lớn để bé dễ nhìn

Màn hình:
1. Welcome screen
2. Chọn level
3. Gameplay
4. Reward screen
5. Result screen

Reward system:
- Tặng sao sau mỗi câu đúng
- Sticker reward
- Badge hoàn thành level
- Mở khóa theme mới

Theme system:
- Candy land
- Ocean world
- Space adventure
- Farm world
- Dinosaur land

Mascot:
- Có mascot dễ thương hướng dẫn bé chơi
- Mascot phản ứng:
   - vui khi đúng
   - buồn nhẹ khi sai
   - clap animation

UX requirements:
- Responsive mobile/tablet
- Button lớn cho trẻ em
- Không có quảng cáo
- Bé có thể tự chơi
- Touch interaction mượt
- Không cần keyboard

Bonus features:
- Hệ thống lưu tiến trình bằng localStorage
- Daily reward
- Progressive difficulty
- Animation ngày/đêm
- Particle background
- Voice recognition đọc đáp án
- Chế độ multiplayer với phụ huynh

Yêu cầu coding:
- Viết toàn bộ source code hoàn chỉnh
- Tạo UI đẹp như game trẻ em thực tế
- Có animation CSS/canvas
- Dùng SVG hoặc emoji miễn phí
- Có cấu trúc project rõ ràng
- Có reusable component
- Tối ưu hiệu năng cho mobile
- Không dùng asset trả phí

Mục tiêu cuối cùng:
- Trải nghiệm giống ứng dụng học toán premium cho trẻ em
- Bé cảm thấy vui và muốn chơi nhiều lần
- Học phép cộng thông qua tương tác trực quan và animation
