# Number Pop

## Overview
A fun educational mini-game for children aged 2–6 years old.

Children learn:
- Number recognition
- Counting
- Basic math
- Focus and reaction skills

Core gameplay:
- Numbers appear as floating balloons/bubbles.
- The game asks the child to tap/pop the correct number.
- Correct answers trigger exciting animations and sounds.

The experience should feel:
- Energetic
- Rewarding
- Easy to understand
- Fast and satisfying

---

# Core Gameplay

## Main Concept

The screen shows:
- Multiple floating balloons
- Each balloon contains a number

Voice instruction:
> “Pop number 3!”

The child taps the correct balloon.

# Game Flow

## 1. Start Round
- Balloons float upward slowly
- Random numbers appear
- Friendly mascot gives instruction

Example:
> “Can you find number 5?”

---

## 2. Child Taps Balloon

### Correct Answer
Trigger:
- Balloon pop animation
- Star particles
- Sound effects
- Mascot celebration
- Score increases

### Wrong Answer
Trigger:
- Balloon wiggle animation
- Soft “Oops!”
- No punishment
- Encourage retry

---

## 3. Next Round
- New numbers appear
- Difficulty increases gradually

# Emotional Design

The child should feel:
- Excited to tap
- Proud after success
- Motivated to continue

The game should avoid:
- Punishment
- Loud negative sounds
- Frustration

Focus on:
- Positive reinforcement
- Celebration
- Curiosity

---

# Session Design

Each play session:
- 30–60 seconds
- Quick and replayable
- High interaction frequency

Ideal for:
- Short attention spans
- Repeated learning
