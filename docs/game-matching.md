Hãy tạo một mini game web giáo dục trẻ em từ 2 - 6 tuổi theo phong cách puzzle/matching game giống ứng dụng học tập cho trẻ em trên mobile/tablet.

Mục tiêu:
- Giúp bé phát triển:
  - tư duy logic
  - nhận diện hình dạng
  - nhận diện object
  - kỹ năng quan sát
  - phối hợp tay và mắt
- Gameplay đơn giản để bé có thể tự chơi

Ý tưởng gameplay:
- Bé sẽ kéo các mảnh ghép vào đúng vị trí để hoàn thành hình ảnh
- Có nhiều dạng puzzle:
  1. Ghép nửa còn thiếu của con vật
  2. Ghép hình dạng đúng vào object
  3. Match object với shadow/outline
  4. Match object với phần còn thiếu
  5. Kéo shape đúng vào lỗ trống

Ví dụ:
- Con chim bị thiếu một nửa → bé kéo đúng nửa còn lại vào
- Quả dưa có lỗ hình ngôi sao → bé kéo shape ngôi sao đúng vào
- Match hình lục giác với object phù hợp
